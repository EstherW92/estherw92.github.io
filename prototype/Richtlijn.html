Consistent navigation helps people getting confident using Windows 8.  And also ensures better user experience for the blind and visual impaired. They really benefit from consistency and regularity. It allows them to get familiair with an app and the don't need to search for everything over and over again.

How?

Consistent use of navigation

 
 
 
 

Get inspired! >

 

Top app bar

The top app bar is great for switching between different pages of your app. This is mostly used in apps with the flat navigation pattern. This app bar should show up at every page and helps users to navigate easily

 

Header menu

The header menu contains a link to each section page as well as a link back to the hub. This enables users to move around the app quickly. The menu appears at each level and on every page of the app, making it an efficient and reliable way for users to get where they want to go.

 

On-canvas

Use on-canvas controls for filtering, pivoting, or sorting when finding an item is a primary task, like in a collection or search result page. Controls should go into the app bar if the focus of the app is on browsing for content, as with a magazine or shopping app.

 

 

Bottom app bar  

Use the bottom app bar for commands that are related to the current page. Don't use the app bar for navigating from one place in the app to another. All app bar items should act on the content currently in view.

Always start with the commands on the right side of the app bar. Commands which are shown when an item is selected, like delete or pin to start always appear at the left side of the app bar. When you already have some commands at the left, separate them from each other using a separator.
